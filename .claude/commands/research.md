# Investiga un tema en la web

Este comando realiza una investigación profunda sobre el tema especificado usando Tavily API.

## Uso

/research [tema]

## Descripción

1. Busca información actualizada sobre el tema
2. Extrae las fuentes más relevantes
3. Genera un resumen con las fuentes consultadas

## Ejemplos

/research tendencias de IA 2026
/research mejores herramientas de desarrollo web
/research programación en Python

## Notas

- Usa Tavily API para búsquedas
- Guarda resultados en cache por 24 horas
- Máximo 5 fuentes por búsqueda
