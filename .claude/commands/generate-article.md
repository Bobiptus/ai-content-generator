# Genera un artículo completo sobre el tema especificado

Este comando genera un artículo completo usando IA con búsqueda web opcional.

## Uso

/generate-article [tema]

## Ejemplos

/generate-article inteligencia artificial
/generate-article python para beginners
/generate-article mejores prácticas de seguridad

## Opciones

El sistema te preguntará:
1. Tema del artículo
2. Si quieres búsqueda web
3. Tono (profesional, casual, técnico)
